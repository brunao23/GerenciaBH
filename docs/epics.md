# GerenciaisBH - Epics & Stories

Este arquivo é utilizado para gerenciar as features e direcionamentos em andamento do projeto.

## Epic 1: Estabilização do Painel Admin e AI Agents
- **Story 1.1:** Completar migração da configuração de AI para uma página própria no Painel Admin (com persistência multi-tenant).
- **Story 1.2:** Estabilizar sistema de interface de conversa (resolvendo erros 404/MIME em produção).
- **Story 1.3:** Controlar comportamento e velocidade de digitação do agente pela interface frontend.

## Epic 2: Follow-up Adaptativo e Automação
- **Story 2.1:** Configurar Adaptive Tenant Follow-Up Logging (obedecendo horário comercial individual do `units_registry`).
- **Story 2.2:** Integrar e processar Webhooks do IAGOFlow corretamente via automações.

## Epic 3: Otimizações de Banco e Infraestrutura
- **Story 3.1:** Limpeza e normalização de dados multi-tenant de Leads e contatos cruzados.
- **Story 3.2:** Finalizar migração de arquitetura Local Media Factory para processamento em Cloud de Inteligências Artificiais.
